### LMS-UI

LMS Front End

#### Getting Started

1. Clone this repository

    ```bash
    git clone https://github.com/codincity123/LMS-UI.git
    ```

2. Go to project directory and install all dependencies

    ```bash
    cd LMS-UI
    npm install
    ```

3. Available scripts

    ```bash
    # Run development server
    npm run dev

    # Build project
    yarn run build

    # Serve project
    yarn run preview
    ```
